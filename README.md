##项目介绍
* 该项目实现了对用户的管理，角色管理，资源(权限)管理，以及对用户进行分配角色，对角色分配资源(权限)。
* 采用shiro来做权限验证，采用Druid来做数据库的监控
* 基本框架spring+springmvc+spring-jpa，几乎用的都是spring。
* 前端还是使用的jsp页面


##技术选型
* 容器框架:spring
* mvc:spring mvc
* 持久层:spring data jpa+hibernate
* 数据库:mysql
* 数据库连接池、监控:druid
* 安全框架:shiro
* json框架：fastjson
* 日志系统：门面slf4j,实现log4j
* 代码构建工具：maven
* 版本控制：git
* excel解析:jxl
